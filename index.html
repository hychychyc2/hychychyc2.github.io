Description 现代人在生活中面临着巨大压力，高房价、低生育率、996工作制等社会现实，而拖延症、自我轻视、容貌焦虑、失业恐惧、消极情绪、过高期待等常见心理也让本不平坦的生活雪上加霜。在一系列压力的叠加之下，年轻人失去了本该有的锐气和朝气，变得暮气沉沉。一些人因为一件小事而瞬间崩溃，一些人工作 996，生病 ICU，焦虑及抑郁情绪常有，想要与朋友交流发泄情绪时会面临或缺少空闲时间、或没有能交心的朋友，想寻求专业的帮助却又难保不会落入伪科学的圈套。一方面国人对情绪心理问题缺乏正确的认知、逃避甚至有抗拒心理；另一方面国内目前心理体系的鱼龙混杂，缺乏合适的机构、缺乏专业的人士。而作为出现情绪、心理问题人们的身边，家人、朋友，无法及时发现并提供有效的帮助，也是一大重要问题。针对上述存在的问题，我们希望提供一个解决方案，能够让人们正确认识情绪心理问题，能够让社会、人们及时发现存在的心理问题，一起解决问题；能够帮助人们调整情绪，宣泄情绪，让社会焕发生命力。
Project Website hychychyc2.github.io

statement：

我们自我反思并且收集了很多用户对UI界面的意见与满意度，并对设计的界面做了批判，得到了很多问题

问题：

  • 界面类问题

    1. UI界面方框可以把界面沾满很好，但是有的方框改成圆框可能更好

    2. 你有考虑过置顶的位置放到顶部吗

  • 功能问题

    1. 你这个设计的功能非常多样，但是你有没有觉得有点太复杂与混乱了呢（关键）

  • 关怀问题

    1. 你有考虑过可以把一些恐惧心理强迫症这种词去掉吗（关键）

解决：

  • 1.2.来自普通用户，优化界面设计，尽量符合大众美感，尽量减少框的尖锐度，使他更加光滑

  • 3.来自自我组内反思，尽量优化使用体验，减少复杂不必要功能，增加有用的功能，使得用户更加方便使用且能满足用户需求

  • 4.来自病人用户，适当减少敏感词汇，减少用户的恐慌

提高：

  1. 界面设计更加优化，更符合大众美感。
  2. 功能更加完善，减少了冗余，更加方便使用，更能满足用户需求。
  3. 更加亲和，让用户舒服

problems

Posted on 2022-11-04
  1. 根据之前调查数据可知，受众在进行心理咨询时最看重的是隐私性，故我们可以将隐私性作为自己的特色，包括但不限于允许用户匿名咨询、根据用户要求不留存档、设置功能加密等
  2. 针对这一问题，可以从两个方面进行解决:对应用进行主题设计，提供尽可能多的主题，让用户使用自己觉得舒服的主题；设计能体现用户心理的小游戏，通过对游戏中用户的各种选择与行为的分析，帮助咨询师更好的设计咨询过程中的问题和话术
  3. 利用大数据安全手段，为应用添加安全性保障，防止被攻击和信息窃取
  4. 实施”三包政策”，给用户提供试用期和时效较长的售后服务，保证产品是可溯源的
  5. 可以将心理咨询的说法改为提高情绪价值或提升自身幸福感、愉悦度等更容易被人接受的说法，吸引并不是”元气满满”的人们去心理咨询
  6. 做好宣传与普及工作，让有需要的人不再认为精神方面的治疗是难以启齿的事情，让心理健康的人也不再带有色眼镜去看待心理治疗这一事情
